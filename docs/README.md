# Описание проекта
ResumeBuilder — это мобильное приложение для создания профессиональных резюме с использованием готовых ATS-оптимизированных шаблонов и контекстных подсказок. Проект включает в себя разработку прототипа приложения и сопроводительного веб-сайта на GitHub Pages.

## Цель
Создать удобный и функциональный инструмент для составления резюме, адаптированный под потребности современных соискателей, и продемонстрировать процесс разработки через веб-сайт проекта.

## Задачи
- Разработать прототип мобильного приложения с пошаговым конструктором резюме.
- Создать адаптивный веб-сайт проекта на GitHub Pages с описанием этапов работы.
- Реализовать дизайн-систему приложения и обеспечить корректную работу навигации сайта.

## Технологии
- HTML5, CSS3, JavaScript
- Git, GitHub Pages
- Figma (для проектирования интерфейсов)
